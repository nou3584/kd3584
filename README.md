# KINGDOM 3584 Website

This package contains a responsive dark fantasy / candlelit recruitment website.

## Main files

- `index.html`
- `assets/css/style.css`
- `assets/js/main.js`
- `assets/images/*.JPG`

## GitHub Pages

GitHub Pages can publish static files such as HTML, CSS, JavaScript, and images.

PHP, Python, Java, and C++ are included as auxiliary files, but they do not execute directly on GitHub Pages.

## Optional files

- `api/status.php`  
  Optional PHP endpoint for Sakura Rental Server or another PHP server.

- `tools/python/optimize_images.py`  
  Local image optimization helper.

- `tools/cpp/glow.cpp`  
  Optional future WebAssembly source.

- `tools/java/KingdomDataGenerator.java`  
  Optional Java data-generation helper.

## How to publish

1. Create a GitHub repository.
2. Upload all files in this folder.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Select `main` and `/root`.
6. Open the displayed GitHub Pages URL.
